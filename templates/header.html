Εφαρμογές στον Παγκόσμιο Ιστό

Διδάσκων: Βασίλης Βασσάλος

Φοιτητές: Κλειδής Νικόλαος - Σερενίδης Δημήτριος

Ανάπτυξη web εφαρμογής που εκμεταλλεύεται τις σημαντικότερες τελευταίες web τεχνολογίες, με ιδιαίτερη έμφαση σε τεχνικές AJAX, responsive design και σε τεχνολογίες για mobile συσκευές. Βασικό αντικείμενο αποτελεί η ανεύρεση κοντινών σημείων ενδιαφέροντος. Χρησιμοποιούνται δημοσίως διαθέσιμα γεωγραφικά δεδομένα (με πηγές geodata.gov.gr και foursquare.com), διαχωριζόμενα σε κατηγορίες.

Βασικές τεχνολογίες
PHP, html5, css, Javascript, MySQL
Frameworks & apis
JQuery, Google maps api, Handlebars
Τεχνικές Προδιαγραφές
Client & server side validation, Ajax, JSON, token authentication, responsive design, local storage, gzip compression, caching, CSS3, geolocation
