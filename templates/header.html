Εργασία μαθήματος Εφαρμογές στον Παγκόσμιο Ιστό

Διδάσκων: Βασίλης Βασσάλος

Φοιτητές: Κλειδής Νικόλαος - Σερενίδης Δημήτριος

Ανάπτυξη web εφαρμογής που εκμεταλλεύεται τις σημαντικότερες τελευταίες web τεχνολογίες, με ιδιαίτερη έμφαση σε τεχνικές AJAX και σε τεχνολογίες για mobile συσκευές. Βασικό αντικείμενο αποτελεί η ανεύρεση κοντινών σημείων ενδιαφέροντος με τη βοήθεια χάρτη. Πρόσβαση στην πληροφορία έχουν μόνο διαπιστευμένοι χρήστες, βάσει μηχανισμών user authentication και session management σε PHP. Το server-side σκέλος εκτελείται σε περιβάλλον Apache, με RDBMS MySQL. Χρησιμοποιούνται δημοσίως διαθέσιμα γεωγραφικά δεδομένα (με πηγές geodata.gov.gr, foursquare.com και Google Places), διαχωριζόμενα σε κατηγορίες. Η εφαρμογή λαμβάνει αυτόματα την τοποθεσία του χρήστη και παρουσιάζει το στίγμα του στο χάρτη (HTML 5 geolocation + Google Maps) προβάλλοντας γύρω του τα κοντινότερα σημεία από τις επιλεγμένες κατηγορίες. Παρέχεται επιπλέον η δυνατότητα αποθήκευσης τοποθεσιών και προτιμήσεων. Ο χρήστης μπορεί επίσης να προβεί σε αξιολόγηση και υποβολή σχολίου για μία τοποθεσία. Η φόρτωση των δεδομένων και το navigation της εφαρμογής βασίζεται αποκλειστικά σε ασύγχρονες κλήσεις AJAX και DOM manipulation. Για λόγους ελάφρυνσης του φόρτου του server επιτελείται και client side validation των φορμών, εκτός από server-side. Προς αποφυγή περιττών roundtrips στο server, πληθώρα δεδομένων αντλούνται από το local storage του browser, το οποίο θα συγχρονίζεται καταλλήλως. Έχει γίνει κατάλληλη παραμετροποίηση του apache server, μέσω εντολών για τα αντίστοιχα modules στο .htaccess αρχείο, για speed optimization (gzip compression και caching) και SEO friendly URLs (mod_rewrite). Το design της εφαρμογής είναι responsive με έμφαση σε φορητές συσκευές (smartphones & tablets). Για λόγους ευχερέστερης συντήρησης του client-side κώδικα, χρησιμοποιήθηκε το templating javascript framework handlebars. Η ανταλλαγή δεδομένων με το server γίνεται σε format JSON, για λόγους καθαρότητας και καλύτερου διαχωρισμού αρμοδιοτήτων server και client. Έχουν οριστεί τέλος τέλος και vendor specific css properties για ευρύτερη υποστήριξη.

Συνοπτικά οι τεχνολογίες που χρησιμοποιήθηκαν:
Βασικές τεχνολογίες
PHP, html5, css, Javascript, MySQL
Frameworks & apis
JQuery, Google maps api, Js templating system (π.χ. handlebars ή angular)
Τεχνικές Προδιαγραφές
Client & server side validation, Ajax, Json, Session Management, responsive design, speed optimization (local storage, gzip compression, caching), CSS3 (pseudo elements, opacity, transitions, gradients, border radius, shadows), cross browser compatibility, HTML5 geolocation
